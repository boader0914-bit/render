# 여기어때 수집 실패 검토 메모

## 현재 실패 원인

- 현재 여기어때 검색 요청은 `goodchoice.kr`에서 `yeogi.com/domestic-accommodations`로 이동한 뒤 HTTP 403으로 차단된다.
- 실패 유형은 검색어 없음, 파서 오류, UI 오류가 아니라 Cloudflare/WAF 차단이다.
- Node.js 직접 요청은 브라우저에서 실행되는 JS 검증, 쿠키, 브라우저 지문, 세션 검증을 통과하지 못한다.

## 수집 가능 방향

1. 사용자 브라우저 세션 기반 수집
   - 단기적으로 가장 현실적인 방식이다.
   - 사용자가 정상 접속 가능한 Chrome/브라우저에서 검색 결과가 열린 뒤 DOM을 읽는 방식이다.
   - CAPTCHA, 로그인, 차단 안내 등은 사용자가 직접 처리해야 한다.
   - 사이트 정책을 지키는 범위에서 화면에 표시된 결과를 기록하는 보조 방식으로 검토한다.

2. 수동 CSV/HTML 가져오기
   - 여기어때 결과를 사람이 열어 확인한 뒤 HTML 저장본, CSV, 복사 붙여넣기 데이터를 앱에 넣는 방식이다.
   - 자동성은 낮지만 차단 리스크가 가장 낮고 검증용으로 적합하다.

3. 스크린샷/OCR 보조
   - 빠른 현황 파악용으로는 가능하지만 업체명, 가격, 광고 여부 오인식 가능성이 있어 최종 데이터로는 부적합하다.

4. 대체 채널 유지
   - 여기어때가 차단될 때는 떠나요/ONDA, 야놀자/NOL, 네이버를 기본 수집 축으로 유지한다.
   - 여기어때는 상태를 `실패/차단`으로 남기고 원인과 대안을 같이 표시한다.

5. 공식/제휴 API 확인
   - 가장 안정적인 방식이지만 영세업체나 초기 스타트업이 바로 확보하기는 어렵다.
   - 단기 실행 전략이 아니라 장기 제휴/영업 옵션으로 둔다.

## 권장 결론

- 단기: 현재 앱에는 여기어때 실패 원인과 수집 방향을 명확히 표시한다.
- 중기: 브라우저 세션 기반 확인 기능과 수동 가져오기 기능을 별도 옵션으로 만든다.
- 장기: 서비스가 커진 뒤 공식/제휴 API 확보 여부를 확인해 조건형 검색 데이터까지 안정적으로 수집한다.

## 2026-06-08 구현 시도

- Codex에서 사용자의 Chrome 세션을 직접 제어하는 시도는 현재 실행 환경 제한으로 실패했다.
- 웹 앱에 `여기어때 보조 수집` 섹션을 추가했다.
- 사용자는 여기어때 검색 페이지를 정상 브라우저로 열고, 추출 코드를 복사해 페이지 콘솔에서 실행한 뒤 나온 CSV를 앱에 붙여넣을 수 있다.
- HTML, 일반 텍스트, CSV를 붙여넣으면 선택한 실행 결과의 여기어때 행만 보완 데이터로 교체한다.
- 원본 플랫폼 CSV에는 병합 결과가 반영되고, 별도 `*_yeogi_manual_import.csv` 파일도 남긴다.

## 2026-06-09 구조 재검토 및 배포 전 테스트 원칙

- 여기어때는 네이버, 야놀자/NOL, 떠나요/ONDA와 같은 핵심 자동 수집 채널이 아니라 `보조 수동 보완 채널`로 둔다.
- 직접 HTTP 크롤링은 Cloudflare/WAF 차단을 정상 실패로 기록하고, 우회 시도는 하지 않는다.
- 수집 우선순위는 `추출 CSV > 정상 브라우저에서 복사한 페이지 텍스트 > HTML 저장/붙여넣기` 순서로 본다.
- 여기어때 입력 해석기는 `scripts/yeogi_import_parser.cjs`로 분리해 서버 병합 API와 테스트가 같은 파서를 사용한다.
- 배포 전에는 반드시 다음 검증을 통과해야 한다.
  - 문법 검사: `npm run check`
  - 여기어때 파서 케이스 테스트: `npm run test:yeogi`
  - 임시 데이터 폴더에서 `/api/yeogi-import` 병합 테스트
- 테스트 케이스에는 최소한 다음을 포함한다.
  - 줄바꿈 없이 붙은 결과 텍스트
  - 쿠폰 적용가와 일반가가 같이 있는 가격
  - `다른 날짜 확인`, `매진`, `예약마감` 등 예약 불가 문구
  - CSV 붙여넣기
  - 추출 코드 자체를 잘못 붙여넣은 경우
- 배포 금지 조건은 다음과 같다.
  - 입력 4건이 4건으로 병합되지 않는 경우
  - 업체명과 지역명이 깨지는 경우
  - CSV 헤더가 아닌 가격 문장을 CSV로 오인하는 경우
  - 병합 API 테스트 없이 Render 배포를 시도하는 경우
